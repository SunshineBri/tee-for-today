# Tee For Today

**Tee For Today** is a single-page front-end website built to showcase the creativity of AI and the power of charitable giving. Each day, a unique AI-generated T-shirt is featured, and proceeds from purchases are split with a spotlighted charity of the day.

## 🌟 Features

- 🔁 **Daily AI-Generated T-Shirt Design**
- ❤️ **Rotating Charity Spotlight**
- 🎨 **Modern Responsive UI with Tailwind CSS**
- 🧠 **Smooth Animations & Scroll Effects**
- 🖼️ **Modal Pop-Up Preview**

## 🚀 Tech Stack

- **React + Vite**
- **Tailwind CSS** for styling
- **JavaScript (ES6+)**
- **Intersection Observer API** for scroll animations
- **Responsive design** with Flexbox & utility classes

## 🎯 Purpose

This project was built as a coding challenge for **Scarlet Creative Software**. It emphasizes strong UI design, responsiveness, and clean, modular React code — while aligning with a meaningful cause.

Charities are cycled using a modulo system on the day index.
